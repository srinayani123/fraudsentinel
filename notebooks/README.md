# Notebooks

This directory is intentionally empty in the repo. Use it for exploratory analysis:

- `01_data_exploration.ipynb` — explore IEEE-CIS distributions, fraud rates per feature
- `02_xgboost_tuning.ipynb` — hyperparameter sweeps before committing to `train_xgboost.py`
- `03_lstm_experiments.ipynb` — try alternative architectures (GRU, transformer encoder)
- `04_agent_prompt_iteration.ipynb` — test different agent prompts side by side

Drop notebooks in here. They're gitignored when in `.ipynb_checkpoints/`.
